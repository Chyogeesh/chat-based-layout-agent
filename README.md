# Chat Based Layout Agent

AI-powered layout transformation system.

## Features

- Chat based editing
- Layout JSON transformations
- Aspect ratio conversion
- Headline movement
- Offer badge updates
- Wireframe preview

## Setup

### Client

cd client
npm install
npm run dev

### Server

cd server
npm install
npm run dev
